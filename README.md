# Inverted Search System

## 📌 Project Overview

The Inverted Search System is a C-based application designed to efficiently index and retrieve words from multiple text files. It uses data structures like hash tables and linked lists to provide fast search results.

## 🚀 Features

* Index multiple files
* Fast word search using hashing
* Displays file names and word frequency
* Handles collisions using chaining
* Dynamic memory allocation

## 🛠️ Technologies Used

* C Programming
* Data Structures (Linked List, Hashing)
* File Handling

## 📂 Project Structure

* `main.c` → Main driver code
* `hash.c` → Hash table implementation
* `file.c` → File handling
* `search.c` → Search functionality

## ▶️ How to Run

```bash
gcc *.c
./a.out
```

## 📊 Output

* Displays indexed words
* Shows file occurrence and count
